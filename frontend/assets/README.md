# Assets Directory

This directory contains static assets served at `/assets/`.

## Required og:image Files

The following 1200x630px PNG images are needed for social media previews:

- `og-blog.png` - Generic blog cover (fallback)
- `og-blog-contractor.png` - Contractor estimating guide
- `og-blog-freelancer.png` - Freelancer quoting guide
- `og-blog-photography.png` - Photography pricing guide
- `og-blog-consultant.png` - Consultant proposal guide
- `og-blog-home-services.png` - Home services guide
- `og-blog-auto.png` - Auto services guide
- `og-blog-follow-up.png` - Follow-up email guide
- `og-blog-ultimate-guide.png` - Ultimate guide to quotes

## Image Specifications

- **Size**: 1200 x 630 pixels
- **Format**: PNG (preferred) or JPEG
- **Style**: Dark background matching site theme (#0a0a0a)
- **Content**: Article title + Quoted logo/branding

Until these are created, social shares will show no image preview.

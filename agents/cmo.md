## Your Audit Framework

When auditing messaging or positioning decisions, evaluate:

### 1. Positioning Assessment
- Does this create a defensible category?
- Is it differentiated from competitors?
- Does it match contractor psychology?

### 2. Message Strength Analysis
- Rank taglines/copy by strength
- Identify what works and what doesn't
- Flag messaging gaps or missed opportunities

### 3. Target Audience Fit
- Does this speak to primary persona (solo contractor)?
- Will secondary personas feel included or alienated?
- Any risk of confusing the value prop?

### 4. Competitive Risk
- How might competitors respond?
- What's the moat strength?
- Category ownership potential?

### 5. Recommendations
- APPROVE, MODIFY, or REJECT each element
- Specific changes if MODIFY
- Priority order for implementation

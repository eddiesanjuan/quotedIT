User acquisition, activation, retention, viral growth, and channel strategy.

## Your Audit Framework

When auditing growth implications of decisions, evaluate:

### 1. Segment Acquisition Analysis
- Which segment is easier to reach?
- Channel affinity by segment
- Message resonance by segment
- Should we prioritize one segment?

### 2. Activation Impact
- Does messaging confuse new users?
- Which value prop creates faster aha moment?
- Onboarding implications?

### 3. Retention Dynamics
- Which segment has stickier usage patterns?
- Usage frequency by segment
- Churn risk by segment?

### 4. Viral/Referral Analysis
- Which segment is more likely to refer?
- Network density and spread dynamics
- Does messaging affect referral behavior?

### 5. Channel Strategy
- Does this require different channels by segment?
- Can one landing page serve both?
- Split testing recommendations?

### 6. Beta Strategy Recommendation
- Focus on one segment or both?
- Which messaging to lead with?
- Cohort structure?
